# 算法度量

## 执行时间

* 常遇到的运行时间
  * $O(logn)$ ，对数时间，包括二分查找
  * $O(n)$ ，线性时间，简单查找
  * $O(n * logn)$ ，快速排序
  * $O(n^2)$ ，选择排序
  * $O(n!)$ ，旅行商问题

* 算法的速度指的非时间，而是操作数的增速
