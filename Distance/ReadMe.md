# 1 距离度量

## 1.1 欧氏距离

​		空间中两点的直线距离(三维空间也一样)。
$$
\sqrt{(x_i-x_j)^2+(y_i-y_j)^2}
$$

## 1.2 标准化欧式距离

​		针对数据各维分量分布不一样，先标准化到均值、方差相等

​		数据标准化(均值0，方差1)：
$$
X^‘=\frac{X-mean}{square}
$$
​		距离表示为：
$$
\sqrt{\sum_{k=1}^n(\frac{x_{1k}-x_{2k}}{s_k})^2}
$$

##  1.3 曼哈顿距离（城市街区距离）

​		坐标在坐标轴上的距离之和。
$$
|x_i-x_j| + |y_i-y_j|
$$

## 1.4 切比雪夫距离

​		两点之间的最少步数(斜边也算一步)
$$
max(|x_i-x_j|, |y_i-y_j|)
$$

## 1.5 闵可夫斯基距离

​		一组距离的定义。
$$
(\sum_i^n|X_i^{(A)}-X_i^{(B)}|^p)^{1/p}
$$
式中，p是一个可变参数。
$$
\begin{cases}
p=1，曼哈顿距离\\
p=2，欧氏距离\\
p=3，切比雪夫距离\\
\end{cases}
$$

## 1.6 汉明距离

两个字符串进行异或计算，统计结果数为1的个数

## 1.7 夹角余弦

几何中用来衡量两个向量方向的差异，机器学习用来衡量样本向量之间的差异。
$$
cos(θ)=\frac{\vec{a}*\vec{b}}{|\vec{a}||\vec{b}|}
$$


## 1.8 卡方距离

​		卡方统计量来衡量两个个体之间的差异性。
$$
X^2(x,y)=\sqrt{\sum_{i=1}^p(\frac{x_i-E(x_i)}{E(x_i)})^2 + \sum_{i=1}^p(\frac{y_i-E(y_i)}{E(y_i)})^2}
$$
式中，E是期望值。

# 2 应用场景

| 应用       | 距离度量                                 |                |
| ---------- | ---------------------------------------- | -------------- |
| 空间       | 欧氏距离                                 |                |
| 路径       | 曼哈顿距离、切比雪夫距离、闵可夫斯基距离 |                |
| 向量差异   | 夹角余弦                                 | 图像相似性判断 |
| 编码差异   | 汉明距离                                 |                |
| 集合近似度 | 杰卡德相似系数与距离                     |                |
| 相关       | 相关系数                                 |                |
| 时间序列   | DTW距离                                  |                |
